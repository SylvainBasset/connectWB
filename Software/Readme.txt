To add this project to Atollic studio :

- Open atollic studio

- Switch workspace to this location (.../connectWB/Software) :
  "File"->"Switch Workspace"

- Import the project :
  . "File"->"Open Project From File System"
  . "import source" : select (.../connectWB/Software)
  . Select "CentralUnit" or "Software/CentralUnit" with import as "TrueSTUDIO project"


- NOTE : The workspace path shall not contain accent !!!
